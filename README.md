## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/caiocezartg/pokereact-v2

# Access
$ cd pokereact-v2

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
